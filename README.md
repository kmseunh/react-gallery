# react-gallery

## 설명

이 프로젝트는 React를 이용한 사진 갤러리 프로젝트입니다.

&nbsp;

## 목표

- 다양한 주제의 사진 30장을 나열하여 제공합니다.
- 사진 클릭시 해당 사진의 추가 정보를 제공합니다.
